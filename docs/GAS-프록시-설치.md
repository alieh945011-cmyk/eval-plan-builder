# AI 서버(GAS 프록시) 설치 안내 — 5분, 클릭 단위

웹앱의 AI 기능(학기 단위 성취수준 생성, 수행평가 추천)은 Google Apps Script(GAS)를 거쳐
Gemini API를 호출합니다. API 키는 GAS 안에만 저장되어 교사들에게 노출되지 않습니다.
설치는 관리자(선생님) 1명이 1회만 하면 되고, 전부 무료입니다.

## 1. Gemini API 키 발급 (2분)

1. https://aistudio.google.com 접속 → 구글 계정 로그인.
2. 왼쪽 메뉴(또는 우측 상단) **Get API key** → **API 키 만들기** 클릭.
3. 생성된 키(AIza…)를 복사해 둡니다. **이 키는 절대 코드·문서·카톡에 붙여넣지 마세요.**

## 2. Apps Script 프로젝트 만들기 (2분)

1. https://script.google.com 접속 → **새 프로젝트**.
2. 편집기에 기본으로 있는 코드를 모두 지우고, 이 저장소의 `gas/Code.gs` 내용 전체를 붙여넣기.
3. 상단 프로젝트 이름을 `평가계획-AI프록시` 등으로 변경 → 저장(Ctrl+S).

## 3. API 키 등록

1. 왼쪽 톱니바퀴 **프로젝트 설정** 클릭.
2. 아래 **스크립트 속성** → **스크립트 속성 추가**.
3. 속성: `GEMINI_API_KEY` / 값: 1번에서 복사한 키 → **저장**.

## 4. 동작 자체 진단 (선택, 30초)

1. 편집기로 돌아와 상단 함수 선택에서 `selfTest` 선택 → **실행**.
2. 첫 실행 시 권한 승인 창이 뜨면: 계정 선택 → 고급 → "…(안전하지 않음)으로 이동" → 허용.
3. 하단 **실행 로그**에 `모델: gemini-… / 응답: …`이 보이면 정상.

## 5. 웹 앱으로 배포

1. 우측 상단 **배포** → **새 배포**.
2. 톱니바퀴(유형 선택) → **웹 앱**.
3. 설정: 다음 사용자 인증 정보로 실행 = **나** / 액세스 권한 = **모든 사용자**.
4. **배포** → 나온 **웹 앱 URL**(https://script.google.com/macros/s/…/exec)을 복사.

## 6. 웹앱에 연결

1. 평가계획 도우미 접속 → "학기 단위 성취수준" 또는 "AI 추천" 화면.
2. **AI 서버(GAS) 연결 설정**을 열고 5번에서 복사한 URL 붙여넣기 → **저장 후 연결 확인**.
3. "연결됨 · API 키 설정 확인"이 보이면 완료. 이 설정은 각 교사의 브라우저에 저장되므로,
   URL을 교무실 메신저 등으로 공유해 각자 1회 입력하면 됩니다.

## 문제 해결

- **"연결 실패"**: 배포 시 액세스 권한이 "모든 사용자"인지 확인. 학교망이라면 다른 망(핸드폰 테더링)에서도 시도해 차단 여부 구분.
- **"GEMINI_API_KEY가 스크립트 속성에 없습니다"**: 3번 다시 확인 (속성 이름 오타 주의).
- **429/모든 모델 실패**: 무료 키 분당 한도(15회) 초과 — 잠시 후 재시도. 신규 구글 클라우드 프로젝트의 키는 무료 한도가 0인 경우가 있어, 이 경우 다른 계정으로 재발급.
- **코드 수정 후**: 반드시 배포 → 배포 관리 → 연필 아이콘 → 버전 "새 버전" 선택 후 재배포해야 반영됩니다.
